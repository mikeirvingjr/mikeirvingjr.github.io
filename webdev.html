Blazor (Server & Web Assembly)

Blazor was learned while working for Hawcroft. We were waiting for the updates Microsoft Maui was going to bring to Xamarin Forms; however, the updates were not going to be production ready for a while. So, this called for using a combination of Blazor Server and WebAssembly so we could use an embedded SQLite database on the client. Of course the database used EntityFramework and was encrypted just in case the database was compromised.
