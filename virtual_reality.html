Unity

Unity is a 2D and 3D gaming engine designed for professional XR development. It supports Windows, MacOS, and WebGL platforms natively, as well as iOS and Android. This support also extends to XR devices like the Meta Quest series.

I have dabbled a bit in XR projects.

Godot

Godot is an open source and game engine that works to create 2D and 3D games. The primary scripting language is GDScript a language similar to python; however, with the MONO framework and later .NET6, C# became supported. And it also works great for XR devices.

I used the Godot engine as an alternative to Unity due to their new price structure. And it seems that Godot can handle the challenge.
