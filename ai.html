Keras

MonoGame is a managed C# framework for developing cross platform games. It works for a multitude of platforms such as Windows,  MacOS,  Linux,  Android,   and iOS to name a few. If licensed, it will work for Xbox, PS4 | PS5, and Nintendo Switch consoles.

I first used MonoGame somewhere around 2010-2012 where I started porting my old Monopoly C++ code I had written for a Software Engineering class for my Bachelor's degree in Computer Science. Back then, the game was written in C++ and OpenGL. MonoGame proved that it was easier to write in C# and let the framework worry about the graphics layer. I was also using the project to learn C#, but that's is for another story.
